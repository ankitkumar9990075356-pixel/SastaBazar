## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SastaBazaar.git
   ```

2. Navigate to the project directory:
   ```bash
   cd SastaBazaar
   ```

3. Install requirements (if any):
   ```bash
   pip install flask
   ```

4. Run the application:
   ```bash
   python app.py
   ```
   Or run the `run.bat` file on Windows.
